# Claude Code マスター講座

Claude Code を最大限使いこなすためのインタラクティブなレクチャーアプリです。

## 使い方

ビルド不要・依存関係ゼロです。`index.html` をブラウザで開くだけで動きます。

```bash
# そのまま開く
open claude-code-lecture/index.html        # macOS
start claude-code-lecture\index.html       # Windows

# またはローカルサーバーで配信
npx serve claude-code-lecture
```

## 内容

全13章のカリキュラム + リファレンス2種で構成しています。

| 章 | テーマ |
|---|---|
| 01 | Claude Codeとは & セットアップ |
| 02 | 基本操作 — 対話・編集・許可 |
| 03 | スラッシュコマンド完全ガイド |
| 04 | キーボードショートカット & 入力術 |
| 05 | CLAUDE.md — プロジェクトの記憶 |
| 06 | 設定とパーミッション管理 |
| 07 | プランモードと拡張思考 |
| 08 | Gitワークフロー統合 |
| 09 | カスタムコマンド・スキル・サブエージェント |
| 10 | Hooks — 決まりごとを自動化する |
| 11 | MCP — 外部ツールへの接続 |
| 12 | ヘッドレスモードと自動化 |
| 13 | ベストプラクティス総集編 |

### 機能

- **理解度チェック** — 各章末にクイズ。全問正解で章クリア
- **進捗管理** — クリア状況を localStorage に保存(サイドバーに進捗バー表示)
- **チートシート** — コマンド・キー操作・設定ファイルの早見表(キーワード検索付き)
- **プレイグラウンド** — 主要コマンドの雰囲気を安全に試せる疑似ターミナル
- **レスポンシブ対応** — スマホでも学習可能

## 技術構成

単一の HTML ファイル(Vanilla JS / CSS)。外部ライブラリ・ネットワークアクセスなし。
